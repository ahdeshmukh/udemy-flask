How is the login implemented?
The login does not require you to provide a username and password. All you need to do is select a role and submit the form. The login functionality will create a user object in local storage that will be used to maintain the login session. This closely resembles a real login.
What this project does not do?
This project does not have any backend code. It does not communicate with any database.
